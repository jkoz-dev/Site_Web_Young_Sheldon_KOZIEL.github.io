Évaluation sur GIT et Bootstrap concernant un site dédié à la série Young Sheldon.
Premier envoi le 14 décembre 2025 à 14h20.
Débuté le 12/12/2025 à 10h50
Pour ce projet, j'ai commencé par créer le repository avec tout mes fichiers dedans que j'ai upload avec GIT via vscode. Par la suite, j'ai employé Bootstrap pour l'élaboration du site.
L'objectif du projet est de mettre à profit nos compétences en GIT et Bootstrap pour concevoir un site web comportant trois pages dédié à une série télévisée.
Il s'agit donc d'un site web adaptable conçu pour l'évaluation sur GIT et Bootstrap. Il est le présentateur de la série Young Sheldon, avec un aperçu des personnages et un calendrier des saisons.
